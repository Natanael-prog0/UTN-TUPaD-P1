## Descripción

Este trabajo práctico tiene como objetivo ejercitar el uso de estructuras de datos complejas en Python sin utilizar objetos, incluyendo:

- Listas, tuplas, sets y diccionarios.
- Operaciones de acceso, modificación e iteración.
- Manipulación de datos ingresados por el usuario.

Cada ejercicio está resuelto en un solo script con comentarios para facilitar su lectura y mantenimiento.

---

## Contenido

1. Creación y modificación de diccionario de precios de frutas.
2. Actualización de precios.
3. Obtención de nombres de frutas desde un diccionario.
4. Agenda telefónica con ingreso y búsqueda de datos.
5. Procesamiento de frases: sets de palabras únicas y conteo de frecuencia.
6. Cálculo de promedios de alumnos a partir de tuplas.
7. Operaciones con sets para listar estudiantes aprobados.
8. Sistema de stock de productos.
9. Agenda semanal con consultas por tuplas.
10. Inversión de diccionario de países y capitales.

---

## Instrucciones

Ejecutar el script en un entorno Python 3. Se recomienda usar un IDE como Visual Studio Code o correr en consola con `python archivo.py`.

Algunos ejercicios requieren ingresar datos manualmente para su ejecución completa.
